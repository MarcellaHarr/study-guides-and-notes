Module I: Working with Sets

Section 1:


1.2
set theory

A set is a collection of objects. The objects in a set are called elements. The roster notation definition of a set is a list of the elements enclosed in curly braces with the individual elements separated by commas.


1.2.1
empty set

The set with no elements is called the empty set and is denoted by the symbol \(∅\). The empty set is sometimes referred to as the null set and can also be denoted by \(\{ \}\). It is also an element of every power set. A finite set has a finite number of elements. An infinite set has an infinite number of elements.


1.2.2
set member & subset


  • “Element of” Example
  • Subset Example

element \(\epsilon\) set

\(x\enspace \epsilon \enspace \{w,x\}\)

\(p\enspace \epsilon \enspace \{q,r,s\}\)

set \(\subseteq\) set

\(\{6,7,8\}\enspace \subseteq \enspace \{1,2,3,...\}\)

\(\{2\}\enspace \subseteq \enspace \{1,2\}\)

The cardinality of a finite set is the number of elements in that set, and is denoted as \(|\thinspace|\). Two sets are equal if they have exactly the same elements.


1.2.3
theory symbols

A number x is positive if \(x > 0\). A number x is negative if \(x < 0\). The number \(0\) is neither positive nor negative, so \(0 \in \mathbb{Z^+}\) and \(0 \in \mathbb{Z^-}\), and a number \(x\) is non-negative if \(x ≥ 0\).


1.2.4
set builder

In set builder notation, a set is defined by specifying that the set includes all elements in a larger set that also satisfy certain conditions.

\(\mathbb{A} = { x \in \mathbb{S} : \mathbb{P}(x) }\)

\(\mathbb{S}\) is the larger set from which the elements in \(\mathbb{S}\) are taken.


1.2.5 / 1.3
universal set

The universal set is a set that contains all elements mentioned in a particular context, and is denoted by the variable \(\cup\). A Venn diagram visual representation of sets using overlapping circles or other closed curves. However, zyBook’s uses rectangles to denote the universal set and oval shapes to denote sets within \(\cup\).



  • Equal Subsets
  • Proper Subsets

set \(\subseteq\) set

Equal sets \(\{ \}\) can be considered subsets \(\subseteq\). Meaning all of a set’s elements must be found in the other set (superset).

\(\mathbb{A}= \{ \mathbb{a, b, c, d, e} \}\)
\(\mathbb{B}= \{ \mathbb{e, b, c, a, d} \}\)

\(\mathbb{B} \subseteq \mathbb{A}\)

set \(\subset\) set

With proper subsets \(\subset\), equality is not possible. Meaning, a set that is a subset \(\subseteq\) with lesser elements than the superset is considered a proper subset.

\(\mathbb{A}= \{ \mathbb{a, b, c, d, e} \}\)
\(\mathbb{B}= \{ \mathbb{e, b, c, a, d} \}\)
\(\mathbb{C}= \{ \mathbb{e, c, a} \}\)

\(\mathbb{C} \subset \mathbb{A}\) or \(\mathbb{C} \subset \mathbb{B}\)

A subset is a set whose elements are all contained within another set. If every element, for example, in set \(\mathbb{A}\) is also an element of set \(\mathbb{B}\), then \(\mathbb{A}\) is a subset of \(\mathbb{B}\). It is denoted by the symbol: \(\mathbb{A} \thinspace \subseteq \mathbb{B}\).

A proper subset is a subset that is strictly contained within another set, meaning it does not include all elements of the larger set. If \(\mathbb{A} \thinspace \subseteq \mathbb{B}\) and \(\mathbb{A} \thinspace \ne \thinspace \mathbb{B}\), then \(\mathbb{A}\) is a proper subset of \(\mathbb{B}\). It is denoted by the symbol: \(\mathbb{A} \thinspace \subset \thinspace \mathbb{B}\).


1.4
set containing

The empty set \(\varnothing\) is not the same as \(\{ \varnothing \}\). The cardinality of \(\{ \varnothing \}\) is one since it contains exactly one element, which is the empty set. A set can contain a combination of numbers and sets of numbers as in:

\(\mathbb{B} = \{ 2, \varnothing, \{1, 2, 3\}, \{1\} \}\)


1.5
power set

A power set is a set of all the subsets, including the empty set and the set itself. For example, the power set of a set \(\mathbb{A}\), denoted \(P\{\mathbb{A}\}\), is the set of all subsets of \(\mathbb{A}\), so, if \(\mathbb{A} = \{1,2,3\}\), then:

\(P\{\mathbb{A}\} = \{ \varnothing, \{1\}, \{2\}, \{3\}, \{1, 2\}, \{1,3\}, \{2,3\}, \{1, 2, 3\} \}\)

Cardinality of a power set
Let \(\mathbb{A}\) be a finite set of cardinality \(n\). Then the cardinality of the power set of \(\mathbb{A}\) is \(2^n\), or \(\mid P\{\mathbb{A}\}=2^n \mid\).



1.6
intersection & union

Set intersection is the operation that produces a set containing only the elements that are common to both sets. If \(A\) and \(B\) are sets, their intersection is the set of all elements that are members of both \(\mathbb{A}\) **and** \(\mathbb{B}\). It is denoted by the symbol: \(\mathbb{A} \cap \mathbb{B}\)


  • Formula
  • Example
  • Note

\(\mathbb{A} \cap \mathbb{B} = \{ x \thinspace \epsilon \thinspace \mathbb{A} \enspace\text{and}\enspace x \thinspace \epsilon \thinspace \mathbb{B} \}\)

This is the formula for the set intersection.

IF

\(\mathbb{A} = \{ 1,3,5 \}\);
\(\mathbb{B} = \{ 3,4,5 \}\);

THEN

\(\mathbb{A} \cap \mathbb{B} = \{ 3,5 \}\);

The union and intersection operations are commutative. That is,

\(\mathbb{A} \cap \mathbb{B} = \mathbb{B} \cap \mathbb{A}\)


Set union is the operation that produces a set containing all elements that belong to either of two sets. If \(A\) and \(B\) are sets, their union is the set of all elements that are in \(\mathbb{A}\), **or** \(\mathbb{B}\), **or** in both. It is denoted by the symbol: \(\mathbb{A} \cup \mathbb{B}\).


  • Formula
  • Example
  • Note

\(\mathbb{A} \cup \mathbb{B} = \{ x \thinspace \epsilon \thinspace \mathbb{A} \enspace\text{or}\enspace x \thinspace \epsilon \thinspace \mathbb{B} \}\)

This is the formula for the set union.

IF

\(\mathbb{A} = \{ 1,3,5 \}\);
\(\mathbb{B} = \{ 3,4,5 \}\);

THEN

\(\mathbb{A} \cup \mathbb{B} = \{ 1,3,4,5 \}\);

The union and intersection operations are commutative. That is,

\(\mathbb{A} \cup \mathbb{B} = \mathbb{B} \cup \mathbb{A}\)


1.7
set operation


  • Set Operations
  • Set Expression
  • Set Defined

\(\mathbb{A} \cup \mathbb{B}; \enspace \mathbb{A} \cap \mathbb{B}; \enspace \mathbb{A} \setminus \mathbb{B}\)

Set operations are mathematical procedures used to create new sets from existing ones. Common operations include union, intersection, and difference, and they follow rules similar to logical operators in propositional logic.

A set expression is a mathematical statement that combines sets using one or more operations to define a new set. Parentheses are often used to clarify order of operations and ensure accurate interpretation.

An example is: \(\mathbb{A} \cup (\,\mathbb{B} \cap \mathbb{C})\,\),

which represents the union of A with the intersection of B and C.

A defined set is when its elements are explicitly or logically described. It can be defined by listing elements, using conditions, or applying set operations.

For example,

\(\mathbb{A} \cap \mathbb{B} \cap \mathbb{C} \cap \mathbb{D}\)

The set above is defined as the set of elements common to all four sets: \(\mathbb{A}, \mathbb{B}, \mathbb{C}, \text{and} \thinspace \mathbb{D}\).

Understand the definition:

We’re told that for each \(i\) in the set of positive integers (\(\mathbb{Z^+}\)), the set \(A_i\) contains all positive integer multiples of \(i\).

Examples:

  • \(A_i\) is the set of all integers (since every number is divisible by 1)

\(A_1 = \{0,1,2,3,4,5,6, \ldots\}\)

  • \(A_2\) is the set of all even integers

\(A_2 = \{0,2,4,6,8,10, \ldots\}\)

  • \(A_3\) is the set of all integers divisible by 3

\(A_3 = \{0,3,6,9,12,15, \ldots\}\)

  • \(A_4\) is the set of all integers divisible by 4

\(A_4 = \{0,4,8,12,16,20, \ldots\}\)

  • \(A_5\) is the set of all integers divisible by 5

\(A_5 = \{0,5,10,15,20,25, \ldots\}\)

\[ What Does \bigcap_{i=1}^5 A_i Mean? \]

This is asking:

“What integers are in every one of these sets \(A_1, A_2, A_3, A_4, A_5\)?”

That means we’re looking for numbers that are divisible by 1, 2, 3, 4, and 5. So, we need to find integers that are multiples of all these numbers at once.

Find the Least Common Multiple

To satisfy all five conditions, a number must be a common multiple of all five integers and to be a multiple of all five integers, a number has to be divisible by their **least common multiple (LCM)**.

  • LCM(1, 2, 3, 4, 5) = 60
  A₁      A₂        A₃        A₄        A₅
         / \       / \       / \       / \
 none   1   2     1   3     2   2     1   5

Then multiply the prime factors, \(2 \cdot 3 \cdot 2 \cdot 5=60\)

So the set contains all integers that are multiples of 60 — that’s the overlap between all five sets!

The intersection of the sets \(A_1\) through \(A_5\) in set builder notation is:

\[ \bigcap_{i=1}^5 A_i = \{ x:x=60k,\thinspace \text{for} \thinspace k \in \mathbb{Z^+} \} \]

“The set of all values x such that x equals 60 times k, where k is any integer.”

1.7.4
multiple set


  • Intersection
  • Union

\(\bigcap^n_{i=0} A_i\)

The intersection of multiple sets is an operation that yields the set of all elements common to every set in a sequence. When given a collection of sets indexed by integers from 1 to n, their intersection is the set of elements that appear in all of the sets from \(A_\imath\) to \(A_n\).

\(\bigcup^n_{i=0} A_i\)

The union of multiple sets is an operation that yields the set of all elements that appear in at least one of the sets in a sequence. When given a collection of sets indexed by integers from 1 to n, their union includes every element from any of the sets \(A_\imath\) through \(A_n\).


1.8
set/symmetric difference


  • Set Difference
  • Note

\(\mathbb{A} - \mathbb{B}\)

Set difference is the operation that produces a set containing elements that belong to one set but **not** to another. If \(\mathbb{A}\) and \(\mathbb{B}\) are sets, the set difference \(A -B\) contains all elements that are in \(\mathbb{A}\) and not in \(\mathbb{B}\).

The set difference operation is not commutative, meaning \(A - B \neq B - A\) in general.



  • Symmetric Difference
  • Note

\(\mathbb{A} \oplus \mathbb{B}\)

Symmetric difference is the operation that produces a set containing elements that belong to exactly one of two sets — in either \(\mathbb{A}\) or \(\mathbb{B}\), but not in both.

An alternate definition expresses it using union and set difference:

\(\mathbb{A} \oplus \mathbb{B}= ( \, \mathbb{A} - \mathbb{B} ) \, \cup ( \, \mathbb{B} - \mathbb{A} ) \,\)

The symmetric difference is commutative, so \(\mathbb{A} \oplus \mathbb{B} = \mathbb{B} \oplus \mathbb{A}\)

1.8.3
set complement operation


  • Complement Operations
  • Alternate Form
  • Example

The set complement operation produces a set containing all elements in a universal set \((\, \mathbb{U} )\,\) that are **not** in a subset \((\, \mathbb{A} )\,\).

It is denoted by:

  • \((\, \mathbb{A}^` )\,\) or \((\, \overline{\mathbb{A}} )\,\) (depending on notation style)

  • Formally:

    • \([\, \mathbb{A}^` = \{ x \in \mathbb{U} : x \notin \mathbb{A} \} \quad \text{or} \quad \mathbb{A}^` = \mathbb{U} - \mathbb{A}]\,\)

📘 Note: This operation **requires** a well-defined universal set \((\, \mathbb{U} )\,\) unlike union or intersection.

You can express the complement using a set difference operation:

\([\, \mathbb{A}^` = \mathbb{U} - \mathbb{A} ]\,\)

This says: take everything in the universe \(( \, \mathbb{U} ) \,\) and remove the contents of \(( \, \mathbb{A} ) \,\).

Let the universal set be \(( \, \mathbb{U} = \mathbb{Z} ) \,\) the set of all integers.

Define:

\([ \, \mathbb{A} = \{ x \in \mathbb{Z} : x \text{ is odd} \} ] \,\)

Then:

\([ \, \mathbb{A}^` = \{ x \in \mathbb{Z} : x \text{ is even} \} ] \,\)

Which is the set of all even integers.



1.9
combining sets



  • Definition
  • Explanation
  • Example

Combining sets refers to the use of multiple set operations — such as **union**, **intersection**, **difference**, and **complement** — in a single expression to create new sets from existing ones.

These operations may be nested or sequenced to define increasingly complex relationships among sets, and are governed by mathematical precedence similar to arithmetic operations.

  • The complement operation requires a clearly defined **universal set** \((\, \mathbb{U} )\,\) against which the contents of a subset \((\, \mathbb{A} )\,\) are negated.

  • Combined operations should be evaluated by grouping parentheses first, then applying the logic of each operator.

  • Set expressions such as \((\, (\, \mathbb{A} \oplus \mathbb{B})\, \cup (\, \mathbb{A} \cap \mathbb{C})\, )\,\) illustrate how **symmetric difference**, **union**, and **intersection** interact.

    • 🧠 Tip: Just as in algebra, parentheses determine the order of evaluation. For example:

      • \((\, (\, \mathbb{A} \cup \mathbb{B})\, \cap \mathbb{C} )\,\) may produce a different result than \((\, \mathbb{A} \cup (\, \mathbb{B} \cap \mathbb{C})\, )\,\)

Let:

  • \((\, \mathbb{A} = \{ \text{even integers} \} )\,\)

  • \((\, \mathbb{U} = \mathbb{Z} )\,\) the set of all integers

Then:

  • The complement of \((\, \mathbb{A} )\,\) is:

    • \([\, \mathbb{A}^` = \{ x \in \mathbb{Z} : x \text{ is odd} \}]\,\)

Now combine operations:

  • \((\, (\, \mathbb{A} \oplus \mathbb{B})\, \cup (\, \mathbb{A} \cap \mathbb{C})\, )\,\) defines a new set composed of:

    • Elements in \(\mathbb{A}\) **or** \(\mathbb{B}\) but **not both**, combined with

    • Elements shared between \(\mathbb{A}\) **and** \(\mathbb{C}\)


1.9.1
set operations


  • Definition
  • Explanation
  • Example

Set operations are mathematical procedures used to create new sets by combining or comparing existing ones. These operations include:

  • Union \((\, \mathbb{A} \cup \mathbb{B} )\,\): All elements in either set \((\, \mathbb{A} )\,\), set \((\, \mathbb{B} )\,\), or both
  • Intersection \((\, \mathbb{A} \cap \mathbb{B} )\,\): Elements common to both sets
  • Difference \((\, \mathbb{A} - \mathbb{B} )\,\): Elements in \((\, \mathbb{A} )\,\) but not in \((\, \mathbb{B} )\,\)
  • Complement \((\, \mathbb{A}^` )\,\): All elements not in \((\, \mathbb{A} )\,\) relative to a universal set
  • Symmetric Difference \((\, \mathbb{A} \oplus \mathbb{B} )\,\): Elements in exactly one of the sets \((\, \mathbb{A} )\,\) or \((\, \mathbb{B} )\,\)

They allow us to define complex relationships and subsets within a broader universal context.

Set operations can be applied to any number of sets. For instance: - \((\, \mathbb{A} \cup \mathbb{B} )\,\): “All elements that belong to \((\, \mathbb{A} )\,\) or \((\, \mathbb{B} )\,\), or both.” \([\, \mathbb{A} \cup \mathbb{B} = \{ x \mid x \in \mathbb{A} \vee x \in \mathbb{B} \}]\,\)

  • These operations are especially helpful in everyday logic. Example:
    If \((\, \mathbb{A} )\,\) is the set of math majors and \((\, \mathbb{B} )\,\) is the set of computer science majors, then:

    • \((\, \mathbb{A} \cup \mathbb{B} )\,\): Students majoring in either math or computer science
    • \((\, \mathbb{A} \cap \mathbb{B} )\,\): Joint majors
    • \((\, \mathbb{A}^` )\,\): Students not majoring in math, assuming the universal set is all students

🧠 Venn diagrams are a powerful visual tool for illustrating these relationships.

Let:

  • \((\, \mathbb{A} = \{ x \in \mathbb{U} : x \text{ is a math major} \} )\,\)
  • \((\, \mathbb{B} = \{ x \in \mathbb{U} : x \text{ is a CS major} \} )\,\)
  • \((\, \mathbb{U} )\,\) = all students at the school

Then:

  • \((\, \mathbb{A} \cup \mathbb{B} )\,\): Students in either major
  • \((\, \mathbb{A} \cap \mathbb{B} )\,\): Students in both majors
  • \((\, \mathbb{A}^` )\,\): Students who are **not** math majors

Visual interpretation:

In a Venn diagram, the union \((\, \mathbb{A} \cup \mathbb{B} )\,\) would be shaded across both circles for \((\, \mathbb{A} )\,\) and \((\, \mathbb{B} )\,\), representing all students who are in at least one of the two sets.


1.10
set identities & laws



  • Set Identities
  • Atomic Sets
  • Membership Tables

Set identities describe relationships between set operations, such as union, intersection, and complement. These identities are analogous to logical equivalences and can often be proved using similar techniques.

📘 These laws hold within Boolean algebra and serve as foundational rules for manipulating sets.

Atomic sets are the basic building blocks from which more complex set expressions are constructed.

  • Each atomic set is usually a clearly defined, named subset within the universal set \((\, \mathbb{U} )\,\).

  • Examples might include:

    • \((\, \mathbb{A} )\,\): students studying mathematics
    • \((\, \mathbb{B} )\,\): students studying computer science
    • \((\, \mathbb{C} )\,\): students in another major

🧠 When working with two or three atomic sets, **Venn diagrams** help visualize relationships.
However, for four or more atomic sets, diagrams may become too complex and require ellipses to represent all possible regions of overlap.

Membership tables help verify set identities by listing all possible combinations of atomic set membership.

  • For each element:

    • Use **1** to indicate inclusion in a set
    • Use **0** to indicate exclusion

This is similar to **truth tables** in logic, allowing us to compare both sides of an identity expression.

Why use them:

  • They provide a systematic way to validate identities, especially when direct proofs or diagrams become too involved.
  • Useful when dealing with three or more sets, where combinations become numerous.

Example layout:

Element A B C Left Side Right Side
x₁ 1 0 1 ✅ ✅
x₂ 0 1 0 ❌ ❌

✅ indicates the element satisfies both expressions
❌ means it does not belong to the set under that condition
